### 字段说明

- **类型**：F(事实) / O(观点) / I(推测) / E(经历)
- **来源日期**：该信息的发布或获取时间，用于判断时效性
- **等级**：A(官方/原始数据) / B(可信媒体/专业机构) / C(社区/二手) / D(无法定位)
- **是否原始来源**：是一手来源还是转述
- **是否交叉验证**：是否有独立来源确认同一信息
- **支持/反对**：该证据支持还是反对核心观点
- **适用边界**：该证据在什么条件下有效
- **目标章节**：计划用于文章的哪个章节
- **核验状态**：pending / verified / failed / not_applicable
- **使用状态**：unused / used_in_draft / used_in_final / dropped
