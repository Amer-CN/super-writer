## 能力范围

super-writer 覆盖维度 1-11、16-18、20，明确排除 12-15、19。

## 与 v0.1 的变化

| 维度 | v0.1 | v0.2 | 变化 |
|------|------|------|------|
| 选题评分 | 无 | HKR + UEPT 两层 | 新增 |
| 素材充分性检查 | 无 | 分维度评估 | 新增 |
| 研究协议 | 基础 | +排斥分析+痛点深度+停止条件 | 增强 |
| 证据地图 | F/O/I/E | +来源日期+适用边界+核验状态+Reader Tension | 增强 |
| 核心观点提炼 | 找核七问+攻核六刀 | +四铲深化 | 增强 |
| 失败退出 | 无 | 7种退出状态 | 新增 |
| 编辑学习 | 基础 | +双向confidence+key复用+Reviewer语义判断 | 增强 |
| 文风画像 | 基础 | 核心8维+可选4维+confidence管理 | 增强 |
| 类比设计 | 四条件 | +四类裂口判断 | 增强 |
| 评分器校准 | 无 | MAE/Spearman/classification+盲测隔离 | 新增 |
| Burstiness诊断 | 无 | 非阻塞诊断+交接humanizer | 新增 |
